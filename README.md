### 依赖

```
git clone https://github.com/AlexeyAB/darknet
vim Makefile

GPU=1
CUDNN=1
OPENCV=1

make -j8

```

记下载到本地的路径为DARKNET_ROOT，把本工程置于DARKNET_ROOT下

### 训练
```
sh train.sh
```

### 测试单张图片
```
sh test.sh
```
### 评估测试集
```
sh evaluate.sh
```
